# vue2-article

通过学习 liulongbin 大佬的教程，完成的文章阅读项目。
这个项目的数据是从黑马的公开接口中获取。

独立的服务器项目在我的仓库中

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
